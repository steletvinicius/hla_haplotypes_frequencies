Etapas para criar uma ferramenta de busca de haplótipos HLA no streamlit

Por que?
    Conhecer os haplótipos HLA mais comuns auxilia o analista responsável por liberar os resultados na detecção de possíveis
    resultados de baixa confiabilidade.


Encontrar os dados:
    Dados obtidos do 17 Workshop de HLA https://17ihiw.org/17th-ihiw-ngs-hla-data/
    Um dos objeticos deste Workshop foi caracterizar haplótipos HLA em alta resolução
    utilizando-se dados de indivíduos não-relacionados (unrelated) e famílias.


Escrever o algoritmo capaz dfe ler e apresentar estes dados ao usuário:
    Linguagem de escolha: python
    Biblioteca: Pandas!

Interface de exibição da ferramenta ao usuário final:
    Vou utilizar o streamlit

    Setup:
    Como Estou usando o Windows virtualizando o Linux Ubuntu via WSL na versão 2
    https://docs.streamlit.io/library/get-started/installation

    As etapas basicamente consistem em:
    Criar um ambiente virtual - Faça isso porque vai te ajudar na etapa de colocação do app em produção
    Ativar o ambiente virtual
    Instalar as bibliotecas necessárias dentro deste ambiente virtual
    Instalar o streamlit
    COmeçar a testar seu código

    Para rodar seu código e simular seu código na prática (navegador) digite no terminal:
    stremalit run nome_do_arquivo_com_algoritmo.py

    O terminal vai exibir dois endereços no formato: http://xxx.xxx.xx.xxx:xxxx

    Copie e cole esse endereço no navegador ou, segurando a tecla Ctrl, clique com o mouse em um dos endereços
    para abrir o aplicativo em um navegador.
    Não feche o terminal, pois ele está rodando um servidor local para atender os comandos do app no seu navegador.


    Daqui em diante, é só ir construindo seu app passo a passo e ir debugando os erros.
